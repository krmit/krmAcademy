# Webbserverprogrammering 1 

>>>>> ![En flicka skickar webbsidor][] Webben är allt! 

Denna kurs är den mest spännande men också den mest krävande kurs som HTSIT ger. Vi kommer att lära oss hur man skapar sin egen sajt på Internet så att du kan förverkliga dina drömmar.

Vi kommer gå igenom hur man sätter upp en webbservern, programmerar och designar en sajt på den och slutligen ska vi kvalitettsäkra och gör den säker mot attacker. Vi kommer använda oss av PHP för att programmera sajten, sql för att hantera databasen, html för att skapa struktur på sidan och css för att få till ett bra utseende.
 
Vi ska lära oss att programmera med PHP och använda det i verkligheten.

## Innehåll

Vi kommer först att lära oss allmänt om hur webbservrar fungerar och vilka olika tekniska alternativ det finns. Sedan kommer vi gå igenom de tekniker du behöver kunna för att skriva din egen sajt.
