# Guide 2

## Använda heltals variabler

Vi ska lära oss använda __int__ för att deklarera en variabel som kan hantera heltal. 

### Uppgift 1

1.  Kompilera programmet. Vad tror du att du kommer att se?
2.  Gör så att programmer skriver ut 3040.

 ```java
 
		int a = 3;

		System.out.println(a);
 ```

### Uppgift 2

1.  Gör så att programmet skiriver ut värdet av 3+5 genom att ändra värder på variablerna.
2.  Testa att ta bort paranteserna krin (a+b) nedan på rad 13. Vad händer? Varför då?

 ```java
 
		int a = 0;
		int b = 0;

		System.out.println("3*5="+(a+b));
 ```
 
### Uppgift 3

1. Se till att programmet skriver ut rätt svar till formerna nedan."
2. Varför blir det olika resultat på formlerna nedan?
3. Gör en tredjeformel nedan. Använd variblerna nedan och kombinera dem hur du vill med "+", "-", "*" och "/". Använd gärna paranteser.

 ```java
 
 		int a = 2;
		int b = 4;
        int c = 6;
        
		System.out.println("a+b*c=");
		System.out.println("(a+b)*c=");
 ```
 

### Uppgift 4

1. Se till att programmet skriver ut rätt svar till formerna nedan."
2. Varför blir det olika resultat på formlerna nedan?
3. Vad kallas den serie av tal som värdena av variablererna nedan är början av? Är matematik viktigt för programmering?

 ```java
 
		int a = 2;
		int b = 3;
        int c = 5;
        int d = 7;
        
		System.out.println("a-b+c=");
		System.out.println("a*b*c=");
		System.out.println("(b+c)/b=");
		System.out.println("a*b/(a*a)=");
		System.out.println("a*b*c*d=");
 ```
