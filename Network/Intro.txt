# Nätverksteknik

Kursen Nätverksteknik  är en introduktions kurs till adminstration av servrar. Den är också ett förkunskapskrav för nätverksteknikkurser i Gymnasieskolan.

Vi kommer att lära oss att hanter olika typer av servrar. En server är ett program som erbjuder tjänster oftast via ett nätverk. Till exempel har vi webbservrar som ger oss websidorna, FTP servrar som hjälper oss dela filer, e-post servrar som förmedlar e-post och många många fler.  Den som sköter om ett datorsystem kallas för adminstratör, systemadminstratör förkortas "sys adm" på engelska och det är det som du ska bli!

Termen Nätverksteknik kan användas för att beskriva olika saker. Det som menas med Nätverksiktet i OSI modellen är hur routrarna bygger ihop IP-nätverk. Det har således inget med serverhantering att göra, även om en server adminstratör givetviss måste förstå hur nätverkstskitet fungerar. Jag kommer i denna kurs därför att undvika använda begreppet "Nätverksteknik" eftersom det är missvisande. Det vi studerar här är server adminstrion och inget annat.
