# Nätverkskomponenter
> OSI, klienter, servar, routrar, omkopplare, nätslutsar, nätverkskort, kablage

Denna labboration ska vi titta på ett nätverks uppbyggnad. Vi ska lära oss om vilka enheter som ingår i ett nätverk och översiktligt beskriva deras funktion och var de återfinns i OSI-modellen. Vi ska särskilt intressera oss för hur en server arbetar i nätverket och vad dess uppgift är. Vi ska med hjälp av några olika program se hur olika enheter på nätverket arbetar.

## Viktiga principer och begrepp

Klienten
: Är ett program som använder sig av en tjänst på internet. 

Server
: Är ett program som erbjuder en tjänst på ett nätverk.

OSI
: Applikationsskikt, Presentationsskikt, Sessionsskikt, 	Transportskikt, Nätverksskikt, Datalänkskikt, Fysisktskikt.

IP-adress
: Din globala adress, som du använder för att kommunicera på Internet och som du fått av den som ger dig Internet.

MAC-adress
: Din lokala adress, som du använder för att kommunicera på ditt lokala nätverk och som är unikt för varje nätverkskort.

ping, tracerout och ip är bra program för att hantera en nätverksanslutning på en dator.
